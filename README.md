## `üßπ todo clean <LIST_TYPE> <INDEX>`

Clean items list

```
USAGE
  $ todo clean <LIST_TYPE> <INDEX>

OPTIONS
   a, all   all lists
   d, done  done list
   t, todo  todo list

DESCRIPTION
  ...
  Example:
  $ todo clean todo
  $ todo clean all
  $ todo clean done 1

ALIASES
  $ todo cl
  $ todo c
```

_See code: [src/commands/clean.js](https://github.com/toti1212/todoclif/blob/master/src/commands/clean.js)_
